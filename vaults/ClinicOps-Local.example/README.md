# ClinicOps-Local 예시 Vault

이 경로는 실제 Vault가 아니라 구조와 메타데이터만 보여 주는 **합성 예시**입니다. 실제 `vaults/ClinicOps-Local/`은 Git에서 제외되며 외부 동기화·LLM·GitHub·Discord로 전송하면 안 됩니다.

- `00_Inbox/`: 비식별 캡처 초안
- `10_Operations/`: 비식별 SOP와 운영 관찰
- `99_Archive/`: 보존이 끝난 비식별 노트

실제 식별 가능 정보가 보이면 이 저장소에 기록하지 말고 로컬 기관 정책에 따른 별도 절차로 처리합니다.
