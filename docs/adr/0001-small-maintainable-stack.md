# ADR-0001: 작은 교체 가능한 기술 스택 채택

**상태:** 승인됨
**날짜:** 2026-07-10

## 맥락

CORCOIDUM OS는 한 명 또는 소규모 유지보수자가 운영한다. 복잡한 데이터베이스, 대규모 에이전트 오케스트레이션, GraphRAG는 초기 가치보다 유지보수 부담과 보안 검토 범위를 크게 만든다.

## 결정

- 공개 웹은 Vite + TypeScript로 정적 빌드한다.
- 지식 원본은 Markdown/MDX와 Obsidian 호환 frontmatter로 유지한다.
- 자동화는 Python으로 시작하고, n8n은 웹훅·스케줄 등 시각적 오케스트레이션 가치가 분명할 때만 쓴다.
- RAG는 승인 원본 allowlist, 안정적 chunk ID, 인용 표시, AI staging을 갖춘 작은 MVP로 시작한다.

## 결과

장점은 검사 가능성, 로컬 이식성, Git 기반 복구, 낮은 의존성이다. 단점은 대규모 검색·협업 기능이 제한된다는 점이다. 데이터베이스나 복잡한 인프라는 규모·성능 문제를 측정해 확인하기 전까지 도입하지 않는다.
